สถานที่ท่องเที่ยวทั้งหมด

รวมข้อมูลสถานที่ท่องเที่ยว เวลาเปิด-ปิด และการเดินทาง ครบจบในที่เดียว

1. ถนนคนเดินเชียงคาน

ถนนคนเดินเชียงคาน หรือ ถนนชายโขง ตั้งอยู่บริเวณถนนชายโขงในตัวอำเภอเมืองเชียงคาน อยู่ห่างจากตัวเมืองเลยประมาณ 47 กิโลเมตร มีลักษณะเป็นตลาดนัดกลางคืนติดถนนริมแม่น้ำโขง ยาวตั้งแต่ทิศตะวันตกจนถึงทิศตะวันออก บรรยากาศบ้านไม้เก่าแก่อายุกว่า 100 ปี

เวลาเปิด-ปิด : เปิดทุกวัน 17.00-22.00 น.

ที่อยู่ : ถนนชายโขง อำเภอเชียงคาน จังหวัดเลย 42110

ที่จอดรถ : มีลานจอดรถวัดท่าคก หรือลานจอดเอกชนใกล้เคียง

Google Map : คลิกเพื่อดูแผนที่

2. ภูทอก (เชียงคาน)

ภูทอก (เชียงคาน) มีลักษณะเป็นภูเขาสูง ตั้งอยู่ริมแม่น้ำโขง เป็นจุดชมวิวทะเลหมอกที่สวยงามที่สุดแห่งหนึ่งในอำเภอเชียงคาน สามารถชมพระอาทิตย์ขึ้นและทะเลหมอกหนาๆ ในยามเช้า และชมวิวแม่น้ำโขงได้แบบพาโนรามา

เวลาเปิด-ปิด : เปิดทุกวัน 05.00-17.00 น.

ที่อยู่ : ตำบลเชียงคาน อำเภอเชียงคาน จังหวัดเลย 42110

ค่าบริการ : รถรับส่งขึ้น-ลง ภูทอก คนละ 20-30 บาท

Google Map : คลิกเพื่อดูแผนที่

3. Skywalk Chiang Khan (ภูคกงิ้ว)

สกายวอล์ค เชียงคาน หรือ สกายวอล์ค ภูคกงิ้ว ตั้งอยู่ที่ ต.ปากตม อ.เชียงคาน จ.เลย เป็นจุดชมวิวแม่น้ำสองสี (แม่น้ำเหืองไหลมาบรรจบแม่น้ำโขง) ทางเดินทำด้วยกระจกใส สูงจากระดับแม่น้ำโขงประมาณ 80 เมตร

เวลาเปิด-ปิด : เปิดทุกวัน 07.00-18.00 น.

ที่อยู่ : บ้านท่าดีหมี ตำบลปากตม อำเภอเชียงคาน จังหวัดเลย

ค่าบริการ : รถรับส่ง+รองเท้าสวม 60 บาท

Google Map : คลิกเพื่อดูแผนที่

4. แก่งคุดคู้

แก่งหินขนาดใหญ่ขวางกลางแม่น้ำโขง ประกอบด้วยหินก้อนใหญ่ ๆ เป็นจำนวนมาก ตัวแก่งกว้างใหญ่เกือบจรดสองฝั่งแม่น้ำโขง มีกระแสน้ำไหลเชี่ยว เหมาะแก่การนั่งเรือชมวิว (แนะนำช่วงหน้าแล้ง ก.พ.-พ.ค. สวยสุด)

เวลาเปิด-ปิด : 08.00-18.00 น.

ที่อยู่ : ต.เชียงคาน อ.เชียงคาน จ.เลย

ที่จอดรถ : มีลานจอดรถกว้างขวาง

Google Map : คลิกเพื่อดูแผนที่

5. อุทยานแห่งชาติภูเรือ

หนาวสุดในสยาม บนยอดภูเรือมีอากาศหนาวเย็นตลอดปี เป็นจุดชมวิวพระอาทิตย์ขึ้นที่สวยงาม มีดอกไม้เมืองหนาวให้ชมมากมาย โดยเฉพาะในช่วงฤดูหนาว

เวลาเปิด-ปิด : 05.00-18.00 น. (จุดชมวิว)

ที่อยู่ : อ.ภูเรือ จ.เลย

ค่าบริการ : ผู้ใหญ่ 40 บาท เด็ก 20 บาท

Google Map : คลิกเพื่อดูแผนที่

6. ภูกระดึง

หนึ่งในขุนเขาที่เป็นตำนานของนักเดินทาง ต้องเดินเท้าขึ้นเขาเพื่อพิสูจน์ความแข็งแกร่ง ชมพระอาทิตย์ขึ้นที่ผานกแอ่น และพระอาทิตย์ตกที่ผาหล่มสัก

เวลาเปิด-ปิด : เปิดฤดูกาล ต.ค.-พ.ค. (ปิดช่วงหน้าฝน)

ที่อยู่ : อ.ภูกระดึง จ.เลย

ค่าบริการ : ค่าเข้าอุทยาน + ค่าลูกหาบ (ถ้าใช้)

Google Map : คลิกเพื่อดูแผนที่

7. สวนหินผางาม

หรือ "คุนหมิงเมืองเลย" เป็นแนวผาหินปูนสูงใหญ่ทอดตัวเป็นแนวยาว สลับซับซ้อนสวยงามดั่งภูเขาในนิยาย มีเส้นทางเดินศึกษาธรรมชาติที่สนุกและตื่นเต้น

เวลาเปิด-ปิด : 08.00-18.00 น.

ที่อยู่ : อ.หนองหิน จ.เลย

ค่าบริการ : รถอีแต๊กนำเที่ยว 20-30 บาท

Google Map : คลิกเพื่อดูแผนที่

8. ภูป่าเปาะ (ฟูจิเมืองเลย)

จุดชมวิวที่สามารถมองเห็นภูหอที่มีลักษณะคล้ายภูเขาไฟฟูจิ ได้รับฉายาว่า "ฟูจิเมืองเลย" ต้องนั่งรถอีแต๊กของชาวบ้านขึ้นไปชมวิว 360 องศา

เวลาเปิด-ปิด : 05.00-18.00 น.

ที่อยู่ : ต.ปวนพุ อ.หนองหิน จ.เลย

ค่าบริการ : ค่ารถอีแต๊ก 50-60 บาท

Google Map : คลิกเพื่อดูแผนที่

9. พระธาตุศรีสองรัก

ปูชนียสถานคู่บ้านคู่เมืองของจังหวัดเลย สร้างขึ้นเพื่อเป็นสักขีพยานแห่งความรักใคร่ปรองดอง ข้อควรระวัง: ห้ามสวมเสื้อผ้าสีแดงเข้าในบริเวณองค์พระธาตุ

เวลาเปิด-ปิด : 08.00-16.30 น.

ที่อยู่ : อ.ด่านซ้าย จ.เลย

ข้อปฏิบัติ : ห้ามใส่ชุดสีแดง

Google Map : คลิกเพื่อดูแผนที่

10. พิพิธภัณฑ์ผีตาโขน

ตั้งอยู่ในวัดโพนชัย อ.ด่านซ้าย จัดแสดงเรื่องราวประวัติความเป็นมาของประเพณีผีตาโขน หุ่นจำลอง และการสาธิตวาดหน้ากาก

เวลาเปิด-ปิด : 08.00-17.00 น.

ที่อยู่ : วัดโพนชัย อ.ด่านซ้าย จ.เลย

ค่าเข้าชม : เข้าชมฟรี

Google Map : คลิกเพื่อดูแผนที่

11. อ่างเก็บน้ำห้วยกระทิง

แหล่งท่องเที่ยวทางธรรมชาติ บรรยากาศเงียบสงบ เหมาะแก่การล่องแพไม้ไผ่กลางน้ำ รับประทานอาหาร และพักผ่อนกับครอบครัว

เวลาเปิด-ปิด : 09.00-18.00 น.

ที่อยู่ : อ.เมือง จ.เลย

กิจกรรม : ล่องแพ, ทานอาหาร

Google Map : คลิกเพื่อดูแผนที่

12. วัดเนรมิตวิปัสสนา

วัดสวยตั้งอยู่บนเนินเขา วิหารสร้างด้วยศิลาแลงทั้งหลัง สวยงามและวิจิตรบรรจง ภายในประดิษฐานพระพุทธชินราชจำลอง

เวลาเปิด-ปิด : 06.00-18.00 น.

ที่อยู่ : อ.ด่านซ้าย จ.เลย

Google Map : คลิกเพื่อดูแผนที่

13. เขตรักษาพันธุ์สัตว์ป่าภูหลวง

เจ้าของฉายา "มรกตแห่งอีสาน" โดดเด่นด้วยธรรมชาติที่สมบูรณ์ ป่าไม้ร่มรื่น และกล้วยไม้ป่านานาชนิด (โดยเฉพาะกุหลาบขาว) อากาศเย็นสบายตลอดปี

เวลาเปิด-ปิด : 08.00-16.00 น. (เช็คเป็นรอบ/ฤดูกาล)

ที่อยู่ : อ.ภูเรือ จ.เลย

Google Map : คลิกเพื่อดูแผนที่

14. น้ำตกปลาบ่า

น้ำตกสวยโอบล้อมด้วยธรรมชาติ เหมาะกับการแวะเล่นน้ำและปิกนิก บรรยากาศเงียบสงบใกล้ภูเรือ เดินทางสะดวก

เวลาเปิด-ปิด : 08.00-17.00 น.

ที่อยู่ : บ้านปลาบ่า อ.ภูเรือ จ.เลย
